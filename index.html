Passionné de design, j’ai décidé de compléter le certificat de UX design offert par Google via Coursera ainsi que les certifications offertes sur la plateforme Webflow. Je dévore le web design depuis. Figma est devenu mon nouveau meilleur ami.

J’ai parallèlement appris à coder des interfaces web avec HTML et CSS par simple curiosité. Ce site en est l’exemple (soyez gentil je vous en pris..).

Atout pertinent, je suis motion designer de profession, j’ai donc plusieurs cordes à mon arc. Je maîtrise la suite Adobe comme le fond de ma poche pour le 2D ainsi que Blender/Cinema 4D pour le 3D.

Fait zéro pertinent, mais subjectivement intéressant, mon chat s’appelle Jules et je capote sur la pizza. Voilà, c’est dit.
